這篇文章由 **Decap CMS** 後台編輯，存成 Markdown，push 到 GitHub 後由 **Eleventy（11ty）** 在 build 時編譯成一個獨立的 HTML 頁面。

## 跟動態版的差別

- 動態版：瀏覽器打開時才用 JS 抓 md、現場算 HTML
- 這個 11ty 版：**push 後就先算好純 HTML**，搜尋引擎直接讀得到

不需要資料庫，一樣是 Git 存內容，但多了 SEO 與載入速度的好處。
